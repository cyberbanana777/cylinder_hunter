# Описание проекта
Основа - проект "Печенька"

## Окружение (полигон)
Комната, в ней есть цветные цилиндры, которые стоят на полу
У точки `Дом` висит большой ARUCO-маркер. 

## Функционал
Робот от точки `Дом` поворачивается и смотрит на комнату. Если он видит цилиндры - едет к ближайшему, захватывает его, ищет ARUCO-маркер, едет на точку `Дом`, доехжает, ставит цилиндр, разворачивается и начинает вновь анализ окр. среды на наличие цилиндров.

## Особенности реализации
1. В начальный момент у точки `Дом` робот поворачивается до тех пор, пока не увидит цилинр или ARUCO-маркер. 
2. Учитывая, что все цилиндры одинакового размера, ближайший цилиндр выбирается по размеру рамки.
3. Робот едет в сторону цилиндра, подруливая с PD-регулятором, так, чтобы цилиндр находился по центру кадра (уже есть).
4. Подъехав к цилиндру на расстояние 20-30 см, на нижний уровень отправляется команда захвата цилиндра (уже есть)
5. После захвата цилиндра робот совершает разворот на 180 градусов.
6. После разворота робот поворачивается до тех пор, пока не увидит ARUCO-маркер
7. С помощью ARUCO-маркера робот доезжает до точки `Дом` и там ставит цилиндр каким-то образом (возможно, в коробку)
8. Данные о расстоянии до цилиндра робот получает с сонара
9. Цилиндр распознаётся по цвету (синий, красный, зелёный) и обводится в рамку с помощью OPENCV. 

## Конструкционные особенности
Робот состоит из:
    - Корпус
    - 4 мотора постоянного тока с редукторами
    - Плата-драйвер (ЛАРТ)
    - микроконтроллер (Бузина NANO)
    - одноплатный компьютер Raspberry pi model 3B
    - Камера от моноблока (безымянная) + мачта ~30 см
    - Сонар
    - Линейное звено (Шаговый двигатель + драйвер)
    - Схват (Сервопривод)
    - Кнопка включения питания
    - Кнопка запуска программы


## Особенности межпрограммного взаимодействия
- Информация от программы для камеры (с использованием opencv) передаётся в основную программу управления через DMA   
- Связь между микроконтроллером и компьютером реализована через UART (Serial port)
